# PaperBell MD→Word 格式工具箱 v0.1

**原名**：论文格式终极工具箱 v8.4 / v0.6
**当前版本**：v0.1（重命名后的首个版本，整合原独立宏为模块 7）

## 1 功能概述

Pandoc 从 Markdown 导出 Word 后，中英文混排文档常见格式问题（半角标点、多余空格、表格边框等）。本宏提供 7 个独立模块，一键或选择性修复。

| 模块 | 功能 | 输入编号 |
|------|------|----------|
| 1. 智能括号与引用修复 | 括号全角化、引用方括号空格清理 | `1` |
| 2. 标点符号标准化 | 中文语境半角标点→全角 | `2` |
| 3. 深度空格清理 | 中英文/汉数间空格、百分比空格 | `3` |
| 4. 图片样式统一 | 所有图片应用"图片"样式 | `4` |
| 5. Zotero 引用变蓝 | Zotero 引文字段设为蓝色 | `5` |
| 6. MD 表格一键规整三线表 | 标准三线表、五号字、居中 | `6` |
| 7. 精确空格与引号字体修复 | 关闭亚洲自动间距、清除特殊字符、引号字体统一 | `7` |
| **全部执行** | 1 + 2 + 3 + 4 + 5 + 6 + 7 | `0` |

> 支持组合输入，如 `135` 表示只执行模块 1、3、5。

## 2 使用前必读

1. **务必备份**：宏运行后**无法撤销（Ctrl+Z 不可用）**，请先另存副本
2. **图片样式准备**：使用模块 4 前，确保 Word 样式库中有 **"图片"** 样式（居中、单倍行距）
3. **三线表模块**：会清除所有表格边框后重建三线表，如需保留特殊表格格式请跳过模块 6

## 3 安装方法

1. 打开论文 Word 文档
2. 按 **`Alt` + `F11`** 打开 VBA 编辑器
3. 左侧项目栏空白处**右键** → **插入** → **模块**
4. 将 [[paperbell-md-2-word-VBA]] 中的代码完整粘贴到右侧编辑区
5. 关闭 VBA 编辑器，回到 Word

## 4 使用方法

1. 按 **`Alt` + `F8`**
2. 选择 `PaperBell_MD2Word_VBA`，点击**运行**
3. 弹出控制台菜单，输入对应数字

## 5 各模块详解

### 模块 1：智能括号与引用修复

| 修复项 | 示例 |
|--------|------|
| 图名修复 | `( 图 1` → `(图1` |
| 括号转全角 | 中文后 `(` → `（`，汉字/数字后 `)` → `）` |
| 引用清理 | `[ 1 ]` → `[1]` |
| 数字尾巴 | `2023 )` → `2023)` |

### 模块 2：标点符号标准化

仅修改**汉字周围**的半角标点，英文参考文献不受影响。

| 半角 | → | 全角 |
|------|---|------|
| `,` | → | `，` |
| `;` | → | `；` |
| `.` | → | `。` |
| `:` | → | `：` |
| `?` / `!` | → | `？` / `！` |
| `: `（冒号空格） | → | ` `（空格） |

### 模块 3：深度空格清理

| 清理项 | 示例 |
|--------|------|
| 中英文双向去空 | `Word 文档` → `Word文档`，`方案 A` → `方案A` |
| 汉数去空 | `第 1 章` → `第1章` |
| 百分比去空 | `50 %` → `50%` |
| 中文间去空 | 句号后、全角字符间空格 |
| > **保留** | `1 个`（数字+汉字）的空格以防视觉粘连 |

### 模块 4：图片样式统一

- 遍历所有**嵌入型**和**浮动型**图片
- 统一应用"图片"样式
- 文档中不存在"图片"样式时自动跳过并提示

### 模块 5：Zotero 引用变蓝

- 检测 `ADDIN ZOTERO_ITEM CSL_CITATION` 域
- 将引文显示文字设为蓝色，便于视觉识别

### 模块 6：MD 表格一键规整三线表

| 设置项 | 值 |
|--------|------|
| 顶线 / 底线 | 1.5 磅实线 |
| 表头下分隔线 | 0.75 磅实线 |
| 其他边框 | 无 |
| 字体 | 宋体，五号（10.5pt） |
| 对齐 | 水平居中 + 垂直居中 |
| 列宽 | 100% 页面宽度自适应 |

### 模块 7：精确空格与引号字体修复（v0.1 新增）

整合自原独立宏 `Fix_Spacing_And_Quote_Font`，处理模块 3 通配符方式覆盖不到的细节问题。

**三部分功能：**

**① 关闭亚洲文字自动间距**（段落级属性）

Word 默认在中英文、中数字间自动插入微间距（不是空格字符），导致排版有"缝隙"。逐段落关闭此属性。

**② 清除特殊空格字符**（逐字精准替换）

| 字符 | Unicode | 来源 |
|------|---------|------|
| NBSP | U+00A0 | pandoc-crossref 残留 |
| 全角空格 | U+3000 | 中文输入法误触 |
| ○ · • | — | Pandoc 表格/列表残留 |
| Tab | — | 制表符 |

同时遍历 0~300 的图表式编号空格（`图 1` → `图1`）、引用括号空格（`[ ` → `[`），并反复压缩连续空格。

**③ 引号字体统一宋体**

Pandoc 导出的 `""` `''` 有时继承英文字体，逐个查找并强制设为宋体。

> **模块 3 vs 模块 7 的区别**：模块 3 用通配符批量替换空格，模块 7 用逐字精准替换处理特殊 Unicode 字符、关闭段落属性、修复引号字体。两者互补，建议都执行。

## 7 与 Lua 过滤器的协作

本宏在 Pandoc 导出**之后**运行，处理 Lua 过滤器无法覆盖的 Word 原生格式问题：

```
Markdown
  → Pandoc + pandoc-crossref（交叉引用）
  → Lua: zotero.lua（Zotero 引用字段）
  → Lua: move-tbl-caption-en.lua（表格英文标题上移）
  → Lua: replace-nbsp.lua（NBSP 替换）
  → Word 文档
  → VBA: PaperBell_MD2Word_VBA（括号/标点/空格/图片/三线表/引号字体）
  → 最终文档
```

> **建议执行顺序**：先 `0` 执行全部，再检查是否有遗漏单独补跑。

## 8 常见问题

### Q: 运行提示"编译错误"或代码变红

复制粘贴时可能漏掉了首尾的 `Sub` / `End Sub`，请重新全选复制。

### Q: 图片样式报错

检查 Word 样式库中是否有"图片"样式。没有则新建一个（居中、单倍行距），或修改代码中 `picStyleName` 为已有样式名。

### Q: 英文参考文献的标点被改了吗

**不会**。所有标点替换仅针对汉字周围的半角标点，`Vol. 12, No. 3` 等纯英文/数字组合安全保留。

### Q: 三线表模块误改了特殊表格

模块 6 对文档内所有表格统一处理。如有需要保留特殊格式的表格，请先手动处理完三线表，再删除代码中的模块 6 相关部分。
